DEPLOYMENT STEPS
1. Delete old public website files from the GitHub Pages repo root.
2. Upload the CONTENTS inside sarksh-site-main, not the folder itself.
3. Confirm these files are present: index.html, assets/css/sarksh.css, assets/js/sarksh.js, sitemap.xml, robots.txt.
4. Wait for GitHub Pages deployment.
5. Open https://www.sarksh.in/?v=9 and press Ctrl+Shift+R.
