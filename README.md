# SARKSH GROW Website V16

Premium one-page SARKSH GROW website with SEO, favicon, sitemap, AI-readable summary, responsive UX, learning pages and Google Apps Script lead backend integration.

## Run locally

```bash
npm install
npm run start
```

Then open: `http://localhost:8080`

## Validate

```bash
npm run check
```

## Important

The Apps Script backend URL is preserved in the website files. Upload the full folder contents to GitHub Pages, including the `assets/`, `scripts/`, `sitemap.xml`, `robots.txt`, favicon files and all HTML files.
